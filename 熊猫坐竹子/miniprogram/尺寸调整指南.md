# 熊猫爬竹子游戏 - 尺寸调整指南

## 📐 主要尺寸参数说明

### 1. GameBoy 机身尺寸
**文件：`pages/game/game.wxss`**

```css
.gameboy-body {
  width: 700rpx;   /* 机身宽度 */
  height: 1200rpx; /* 机身高度，建议为宽度的 1.7 倍 */
}
```

### 2. 游戏画面尺寸 ⚠️ 最重要
**文件：`pages/game/game.wxss`**

```css
.lcd-display {
  width: 460rpx;   /* 游戏画面宽度 */
  height: 400rpx;  /* 游戏画面高度 */
}
```

**⚠️ 修改后必须同步更新 JS 文件！**

**文件：`pages/game/game.js`**

```javascript
CONFIG: {
  LOGICAL_WIDTH: 460,   /* 必须与 WXSS 中的 width 一致 */
  LOGICAL_HEIGHT: 400,  /* 必须与 WXSS 中的 height 一致 */
  polesX: [85, 230, 375], /* 竹子位置，建议：[宽度*0.18, 宽度*0.5, 宽度*0.82] */
}
```

### 3. 按钮尺寸

#### 方向键（D-Pad）
**文件：`pages/game/game.wxss`**

```css
.d-pad {
  left: 0rpx;      /* 左边距 */
  top: 20rpx;      /* 上边距 */

  height: 234rpx;  /* 整体高度 */

```

#### A/B 按钮
```css
.action-btns {
  right: 10rpx;    /* 右边距 */

}

.round-btn {
  width: 114rpx;   /* 按钮直径（当前为原始尺寸的 1.3 倍）*/
  height: 114rpx;
}
```

#### SELECT/START 按钮
```css
.function-btns {
  margin-top: 60rpx; /* 上边距 */
  gap: 52rpx;        /* 两个按钮间距 */

.pill-btn {
  width: 130rpx;   /* 按钮宽度（当前为原始尺寸的 1.3 倍）*/

}
```

## 🎯 常见调整场景

### 场景 1：整体放大游戏机
1. 修改 `.gameboy-body` 的 `width` 和 `height`
2. 按比例调整 `.screen-casing` 的尺寸
3. 按比例调整 `.lcd-display` 的尺寸
4. **同步更新** `game.js` 中的 `LOGICAL_WIDTH` 和 `LOGICAL_HEIGHT`
5. **重新计算** `polesX` 的值

### 场景 2：只放大游戏画面
1. 修改 `.lcd-display` 的 `width` 和 `height`
2. **必须同步更新** `game.js` 中的 `LOGICAL_WIDTH` 和 `LOGICAL_HEIGHT`
3. 重新计算 `polesX` 的值

### 场景 3：调整按钮大小
1. 修改对应按钮的 `width` 和 `height`
2. 调整 `gap`（按钮间距）
3. 调整 `left`/`right`/`top`（位置）

### 场景 4：调整游戏角色大小
**文件：`pages/game/game.js`**

在 `draw()` 方法中找到对应的绘制代码：

```javascript
// 熊猫放大倍数
const scale = 1.5;  // 修改此值调整熊猫大小

// 虫子放大倍数
const bugScale = 1.3;  // 修改此值调整虫子大小
```

## ⚠️ 重要提示

1. **同步修改**：修改 `.lcd-display` 尺寸后，必须同步修改 `game.js` 中的 `CONFIG`
2. **比例协调**：建议保持各元素的比例关系，避免变形
3. **测试验证**：每次修改后都要在真机上测试效果
4. **备份代码**：修改前建议备份当前可用的版本

## 🔧 快速调整公式

### 按比例放大整个游戏机
假设放大倍数为 `X`：

```
新机身宽度 = 700 * X
新机身高度 = 1200 * X
新游戏画面宽度 = 460 * X
新游戏画面高度 = 400 * X
新 LOGICAL_WIDTH = 460 * X
新 LOGICAL_HEIGHT = 400 * X
新 polesX = [85*X, 230*X, 375*X]
```

### 竹子位置计算
```
左竹子 X = LOGICAL_WIDTH * 0.18
中竹子 X = LOGICAL_WIDTH * 0.5
右竹子 X = LOGICAL_WIDTH * 0.82
```

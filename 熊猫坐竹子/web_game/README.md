# Panda Bamboo Climber - GameBoy Edition

A web-based simulation of a GameBoy Color game where a panda climbs bamboo to avoid and attack bugs.

## How to Run
Simply open `index.html` in any modern web browser.

## Controls
- **Arrow Keys**: Move Panda Up/Down and Switch Bamboo Poles (Left/Right).
- **Z Key**: B Button (Not used in this game).
- **X Key**: A Button / Slide Attack.
- **Enter**: Start Game / Restart.

## Rules
1. **Movement**: Use Left/Right to jump between the left and right sides of the 3 bamboo poles.
2. **Avoid Bugs**: Bugs climb up from the bottom. Don't let them touch you!
3. **Attack**: Press **X** (or the A Button on screen) to **SLIDE DOWN**. If you hit a bug while sliding, you destroy it and get points!
4. **Health**: You have 3 Hearts.
5. **Difficulty**: Bugs get faster and more frequent as your score increases.

## Design Notes
- **Retro Aesthetic**: CSS-styled GameBoy chassis.
- **Visuals**: Canvas-based pixel-art style rendering with gradient shading.
- **Engine**: Custom JavaScript game loop (60 FPS).
